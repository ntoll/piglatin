Welcome to NHS PigLatin's documentation!
========================================

Welcome to my awesome project. PigLatin is an essential aspect of the NHS's
future plans for cross species communication of a porcine nature.

More arbitrary waffle.

Developer Setup
---------------

Easy peasy::

    $ pip install -r requirements.py

Usage
-----

PigLatin provides a single function called ``translate``::

    from piglatin import translate

    result = translate("Hello world!")
